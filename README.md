# Учебная виртуальная машина (УВМ)

## Описание проекта

Проект представляет собой реализацию ассемблера и интерпретатора для учебной виртуальной машины (УВМ). Разработка выполнена в три этапа, каждый из которых добавляет новую функциональность.

## Структура проекта

Проект разделен на три этапа, соответствующих этапам разработки:

### Этап 1: Базовый ассемблер
- **assembler.py** - основной файл ассемблера с парсингом JSON-программ
- **program.json** - тестовая программа из спецификации УВМ

### Этап 2: Формирование машинного кода
- **assembler2.py** - улучшенный ассемблер с кодированием в двоичное представление
- **spec_test_program.json** - программа для тестов спецификации (пункт 5 этапа 2)

### Этап 3: Интерпретатор
- **interpreter.py** - интерпретатор УВМ с моделью памяти
- **copy_array_program.json** - программа копирования массива (пункт 6 этапа 3)

## Язык ассемблера УВМ

Язык ассемблера использует JSON формат. Программа представляет собой массив команд с операндами.

### Формат программы:
```json
{
  "program": [
    {"command": "LOAD_CONST", "operand": <значение>},
    {"command": "READ_MEM", "operand": <смещение>},
    {"command": "WRITE_MEM", "operand": <адрес>},
    {"command": "DIV", "operand": <адрес>}
  ]
}
